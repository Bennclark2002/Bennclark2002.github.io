Stocks Data Visualizer

This was a group project that I was a part of whose main duty was to show the overall growth or decay of a stock over a certain period. We had it set up so that it was getting info from an API that we had to use to get the information for the stocks. The user was able to choose from an assortment of stocks that were present and could choose from Weekly, Biweekly, Monthly, and Yearly as well as being able to choose from a line graph or bar graph. Once chosen it would display the graph beside the options.
